## Reply Compose Section Details

### Location
Between the message history and the action buttons at the bottom of the modal.

### Components

#### 1. Textarea Input
- **Label**: "Send Reply"
- **Placeholder**: "Type your reply here... (This will be sent directly to the vendor)"
- **Size**: 4 rows tall, full modal width
- **Behavior**: 
  - Grows as you type
  - Auto-disabled while sending
  - Text is preserved until manually cleared

#### 2. Character Count
- **Display**: "Character count: X"
- **Updates**: In real-time as you type
- **Purpose**: Help admin gauge message length

#### 3. Send Button
- **Label**: "ğŸ“§ Send Reply to Vendor"
- **Color**: Blue (#2563EB)
- **Size**: Full width
- **Behavior**:
  - **Disabled** when textarea is empty
  - **Disabled** while reply is being sent
  - **Enabled** when text is present and not sending
  - Shows "Sending..." spinner while active
