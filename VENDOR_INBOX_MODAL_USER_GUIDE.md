### Reading Messages

1. **Click** on a conversation in the left list
2. The **thread opens** on the right
3. **Scroll** to see all messages
4. **Gray bubbles** = Messages from Admin
5. **Blue bubbles** = Your replies

### Replying to a Message

1. **Open** a conversation
2. Find the **compose box** at the bottom (has text input)
3. **Type** your reply
4. **Click the send button** (â†’ arrow) to send
5. Your message appears in **blue** immediately
6. Admin receives your reply in their panel

### Attaching Files

1. In the **compose box**, click the **ğŸ“ paperclip icon**
2. **Select files** from your computer
3. Selected files appear as **chips** above the text box
4. **Click the X** on any chip to remove it
5. **Send the message** - attachments go with it
6. Recipients can **click the attachment** to download

### Filtering Conversations

Click any of these buttons at the top:

- **[All]** - Show all conversations
- **[Unread]** - Show only conversations with new messages
- **[Read]** - Show only conversations you've already read
- **[Archived]** - Show archived conversations

**Active filter** is highlighted in **amber/orange**

### Searching for Conversations

1. **Type** in the search box at the top
2. Conversations **filter in real-time**
3. **Clear** the search to see all again
4. Searches match **message content** (case-insensitive)

## â“ Frequently Asked Questions

**Q: Will I be notified when I get a new message?**  
A: Yes! The Inbox button will show a red badge with the count.

**Q: Can I edit messages after sending?**  
A: Not yet - we're planning this feature. For now, just send a follow-up message.

**Q: How do I know if admin has read my message?**  
A: The message won't have a "seen" indicator yet, but you can see the timestamp when they reply.

**Q: Can I delete individual messages?**  
A: Currently only full conversations can be deleted. Individual message deletion is coming soon.

**Q: What file types can I attach?**  
A: Any file type! PDF, images, Word docs, Excel, ZIP, etc.

**Q: What's the file size limit?**  
A: Up to 100MB per file (limit may vary).

**Q: Can I have conversations with multiple admins?**  
A: Yes! Each admin conversation is separate in your list.

**Q: Will my archived conversations be deleted?**  
A: No! Archiving just hides them. They stay in the [Archived] filter.

**Q: What happens if I close the modal?**  
A: Your message draft is lost (we're adding auto-save soon). But conversations stay saved.
