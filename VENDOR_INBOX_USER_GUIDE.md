## â“ Common Questions

### Q: Why can't I see the Inbox button?
**A:** You must be logged into your own vendor profile. The button only shows for vendors viewing their own account.

### Q: Do messages save automatically?
**A:** Yes! When you click Send, your message is saved immediately. You'll see it in blue in the thread.

### Q: Can I edit messages?
**A:** Not yet. But you can delete the conversation and start fresh if needed.

### Q: What happens when I archive?
**A:** The conversation is hidden from the main list, but you can see it again by switching to the "Archived" filter. Nothing is deleted.

### Q: What happens when I delete?
**A:** The conversation is **permanently removed** - you can't get it back. The admin still has a copy on their side, but you won't see it anymore.

### Q: Are files secure?
**A:** Yes, files are stored in secure cloud storage. Only you and the admin can access them.

### Q: Can I attach any type of file?
**A:** Yes! PDFs, images, documents, spreadsheets, etc. - anything you need to share.

### Q: Will I see messages from the past?
**A:** Yes! When you open a conversation, you see the entire message history, all the way back to the first message.

### Q: Do messages appear in real-time?
**A:** Yes! If the admin sends a message while you're looking at the conversation, it appears immediately without refreshing.

## ğŸ†˜ Troubleshooting

### Modal won't open
1. **Refresh the page** (Cmd+R on Mac, Ctrl+R on Windows)
2. **Check you're logged in** to your vendor account
3. **Verify you're on your own profile** (not viewing another vendor)

### Messages not appearing
1. **Refresh the modal** (close and reopen)
2. **Check your internet connection**
3. **Try a different browser** if issue persists
4. **Contact support** if still not working

### Files won't upload
1. **Check file size** - Very large files may timeout
2. **Check file type** - Some file types not supported
3. **Try a different file** to test
4. **Check internet connection** - Upload needs stable connection

### Can't see admin's name
1. **This is rare** - Most admins have names in the system
2. **It will show "Admin"** as a fallback
3. **Contact support** if this is an issue
